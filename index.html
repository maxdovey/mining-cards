Mining Cards is a collectable trading card series featuring a selection of custom made crypto-currency mining rigs.
Each card displays a custom made mining rig accompanied with the computers specific attributes to play against one another similar to the game Top Trumps.
Each card is stored on a distributed network and contains a unique hash that can be used to validate ownership of each card and verify future editions.

This first edition features x25 ethereum mining rigs made between 2015 - 2017.

  • 1
    The starting player selects an attribute from their first card (e.g. Hashing Rate, Power Consumption or Unique Score).
  • 2
    Each player reads out the value of that attribute on their card and whoever has the highest or best value wins that round.
  • 3
    This continues for as many turns as necessary and players drop out of the game as they run out of cards. Whoever manages to collect all the cards wins the game.

Mining Cards will be released in both digital and print editions. The digital editions will be distributed using ethereum ERC-20 tokens, a protocol for exchanging digital information on the blockchain. Each card is stored on a distributed file storage system (IPFS) and will use an ethereum smart contract to issue a number of editions of each card.
